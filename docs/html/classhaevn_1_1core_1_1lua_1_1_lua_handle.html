Detailed Description

This class handle the LUA interaction, it also provides the LUA UI-Designer API.

Important The LUA UI Designer API is not documented here because it can not be called by the C / C ++ SOURCE code. However, the LUA UI Designer API is stored on GitHub. https://github.com/nimile/MMI-GUI/blob/master/LUA_UI-Designer_API.md
